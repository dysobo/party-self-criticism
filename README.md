# 自我批评参考素材一键生成（后端增强版）

一个基于 AI 的自我批评材料生成工具，支持自定义 API Key，适用于所有岗位。

## 📦 分支说明

### `master` - 静态 HTML 版本
- 纯前端静态页面，无需后端
- 适合简单部署，直接放到 Web 服务器即可
- 不部署双击运行也可以

### `backend` - 后端增强版本（当前分支）
- Node.js + Express 后端服务
- 密码后端验证，前端不存储明文密码
- 5 次错误锁定 IP 30 分钟
- Docker 容器化部署

## 功能特性

### 核心功能
- 🔐 密码验证（后端版本）
- 📝 AI 自动生成自我批评材料
- 🤖 批评他人材料生成
- 📄 导出 Word 文档
- 💾 个人信息导入导出
- 🔄 实时流式输出
- ⚙️ 支持自定义 API Key

### 生成规则
- 严格基于用户填写的工作内容
- 从5个方面查找差距不足：
  - 学习贯彻党的创新理论
  - 加强党性锤炼
  - 联系服务群众
  - 发挥先锋模范作用
  - 改作风树新风
- 内容真实具体，避免假大空
- 语言朴实，避免空洞表态

## 技术架构

### 后端技术
- Node.js + Express
- IP 锁定保护机制
- 密码后端验证
- Docker 容器化

### 前端技术
- 纯 HTML + CSS + JavaScript
- 使用 localStorage 保存用户配置

### API 集成
- 支持多个 AI 模型提供商
- 默认: AiHubMix (glm-4.7-flash-free)
- 推荐: DeepSeek (deepseek-chat)
- 支持自定义 API 配置

### 依赖库
- docx.js: 生成 Word 文档
- FileSaver.js: 文件下载
- express: Web 服务器

## 部署方式

### 方式一：Docker Compose（推荐）

```bash
# 构建并启动
docker-compose up -d

# 查看日志
docker-compose logs -f

# 停止服务
docker-compose down
```

### 方式二：直接运行

```bash
# 安装依赖
npm install

# 启动服务
npm start

# 或自定义密码
PARTY_PASSWORD=your_password npm start
```

访问：http://localhost:3888

## 配置说明

### 环境变量

| 变量名 | 说明 | 默认值 |
|--------|------|--------|
| PARTY_PASSWORD | 密码验证 | xxxxxxxx |
| PORT | 服务端口 | 3888 |

### API 配置

点击左上角"⚙️ API配置"，推荐使用 DeepSeek：
1. 访问 https://platform.deepseek.com 注册账号
2. 获取 API Key
3. 选择 DeepSeek 并填入 API Key
4. 费用: 约 1 分钱/次生成

## 安全特性

1. **密码后端验证** - 前端不存储明文密码
2. **IP 锁定保护** - 5 次错误后锁定 30 分钟
3. **Docker 隔离** - 容器化运行，环境隔离

## 使用说明

### 1. 填写信息
- 姓名
- 岗位
- 主要工作内容（越详细越好）
- 自定义重点内容（可选）
- 上传去年文稿（可选）

### 2. 配置 API（推荐）
点击左上角"⚙️ API配置"配置自己的 API Key

### 3. 生成和导出
- 点击"一键生成"
- 可编辑生成结果
- 导出为 Word 文档
