## 技术架构

### 前端技术
- 纯 HTML + CSS + JavaScript
- 无需后端服务器
- 使用 localStorage 保存用户配置

### API 集成
- 支持多个 AI 模型提供商
- 默认: AiHubMix (glm-4.7-flash-free)
- 推荐: DeepSeek (deepseek-chat)

### 依赖库

## 核心功能说明

### 1. 信息填写
- 姓名
- 岗位
- 主要工作内容（越详细越好）
- 自定义重点内容（可选）
- 上传去年文稿（可选）

### 2. AI 生成规则
生成的自我批评材料必须：
1. 严格基于用户填写的工作内容
2. 从5个方面查找差距不足：
   - 学习贯彻党的创新理论
   - 加强党性锤炼
   - 联系服务群众
   - 发挥先锋模范作用
   - 改作风树新风
3. 内容真实具体，避免假大空
4. 语言朴实，避免空洞表态
5. 字数 800-1200 字
6. 结尾署名和日期（2026年3月）

### 3. 个人信息管理
- 导出个人信息为 JSON 文件
- 导入已保存的个人信息
- 快速填充表单

### 4. API 配置
- 支持切换不同模型提供商
- API Key 本地加密保存
- 默认 Key 不显示，保护隐私

### 5. 结果处理
- 实时流式显示生成内容
- 可编辑生成结果
- 一键导出为 Word 文档
- 重新生成功能

## API 配置说明

### DeepSeek 配置（推荐）
1. 访问 https://platform.deepseek.com 注册账号
2. 获取 API Key
3. API 地址: https://api.deepseek.com/chat/completions
4. 模型名称: deepseek-chat
5. 费用: 约 1 分钱/次生成

### AiHubMix 配置（默认）
1. API 地址: https://aihubmix.com/v1/chat/completions
2. 模型名称: glm-4.7-flash-free
3. 内置免费 Key（已被薅秃）

### 自定义配置
- 支持任何兼容 OpenAI API 格式的服务
- 需要提供完整的 API 地址和模型名称

## 错误处理

### API 调用失败
- 显示详细错误信息
- 检测额度不足时提示申请 DeepSeek
- 引导用户配置 API Key

### 常见问题
1. **生成结果为空**: API Key 额度不足或配置错误
2. **流式输出中断**: 网络问题或 API 限流
3. **Word 导出失败**: 浏览器兼容性问题
